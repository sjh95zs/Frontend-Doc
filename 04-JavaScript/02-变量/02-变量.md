# 变量

## 1、标识符

例如：变量名、函数名、属性名都是属于标识符

## 2、变量与常量的区别

常量：数字和字符串（很少直接用常量）
变量：**用来保存数据的容器**

## 3、变量的声明与赋值

```js
var a = 100,
  b = 200,
  c = 300;
```

> 不写 var，默认为全局变量。**【务必加上 var】**

## 4、命名规范

- 只能由英语字母、数字、下划线、美元符号\$构成
- 只能以字母或是下划线“\_”或者“\$”为开头
- 大小写敏感
- 不能是**关键字**和**保留字**
- 驼峰命名规则（**构造函数**用大驼峰，其它用小驼峰）
- 命名尽量是**有意义的**
