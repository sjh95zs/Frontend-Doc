# JavaScript 的内置对象：

| 内置对象  | 对象说明       |
| :-------- | :------------- |
| Array     | 数组           |
| String    | 字符串对象     |
| Date      | 日期时间       |
| Math      | 数学对象       |
| Number    | 数值对象       |
| Boolean   | 布尔对象       |
| Function  | 函数构造器     |
| Object    | 基础对象       |
| RegExp    | 正则表达式对象 |
| Arguments | 函数参数集合   |
| Error     | 异常对象       |
| Global    | 全局对象       |
| JSON      | 全局对象       |
| ......    | ......         |

**注意**：Math 和 Global 不需要再去实例化，直接使用；而其他则需要通过 new 来创建所需的实例对象。
关于 Math 和 Global 详情：https://segmentfault.com/a/1190000007349910
